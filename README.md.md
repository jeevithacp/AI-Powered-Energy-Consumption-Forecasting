# AI Energy Consumption Forecasting

## Overview
This project predicts energy consumption using Machine Learning (Random Forest Regressor). It analyzes time-based patterns and forecasts future energy usage.

## Tech Stack
Python, Numpy, Pandas, Scikit-learn, Matplotlib

## How to Run
pip install -r requirements.txt
python main.py

## Output
- RMSE score: 4.5
- R2 score: 0.95
- Graph saved in images/
